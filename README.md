# Мой IT-блог на Django

Простой веб-сайт для ведения блога, созданный в рамках производственной практики.

## Как запустить проект:

1. **Установите Python** с сайта python.org
2. **Установите Django** (в командной строке): pip install django
3. **Скачайте проект** или скопируйте файлы в папку
4. **Перейдите в папку проекта** (где лежит manage.py) в командной строке
5. **Запустите сервер**: python manage.py runserver
6. **Откройте браузер** и перейдите по адресу: http://127.0.0.1:8000/

## Как зайти в админку:

1. **Создайте суперпользователя**: python manage.py createsuperuser
2. **Введите логин и пароль** при запросе
3. **Перейдите по адресу**: http://127.0.0.1:8000/admin/
4. **Войдите** с созданными логином и паролем

## Как добавить статью:

1. Зайдите в админку (см. выше)
2. Нажмите "Add" рядом с "Posts"
3. Заполните заголовок и текст статьи
4. Нажмите "Save"

## Как посмотреть статью:

1. **Перейдите по адресу**: http://127.0.0.1:8000
